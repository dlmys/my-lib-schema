# my-lib
Vite base library schema.

## Project setup
```
npm install
```

### Compiles and hot-reloads for development
```
npm run dev
```

### Compiles and minifies for library
```
npm run build:lib
```

### Compiles and minifies for docs
```
npm run build:docs
```

### Preview docs
```
npm run preview
```
